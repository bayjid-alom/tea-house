## 🍵 Tea House

### A modern and responsive tea shop landing page built with HTML and Tailwind CSS.

🌐 Live Demo

[Visit : Tea House](https://bayjid-alom.github.io/tea-house/)

### 📖 About The Project

Tea House is a responsive frontend website designed for a premium tea brand. The project showcases featured tea products, customer testimonials, and the latest news in an elegant and user-friendly interface.

The main goal of this project is to practice responsive web design, modern layouts, and Tailwind CSS fundamentals while creating a visually appealing landing page.

## ✨ Features

* Responsive Design
* Modern UI Layout
* Hero Banner Section
* Clean Code Structure
* Mobile-Friendly Interface
